# Linguee to Anki card adder

This extenion can add words on Linguee directly to Anki. Cross-browser. Early beta app.

## Features

- Add words to Anki directly from Linguee with one click.
- Retain all information including word class (noun, verb, adjective, etc.), gramatical gender 😱, translations, examples, 
- Fully offline. No cloud or server needed. No data collected.
- Cross-browser. Works on Chrome (and most probably Edge, Brave, Opera, etc) and Firefox.
- Open source. Code available on GitHub.
